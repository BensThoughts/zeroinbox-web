• Open the menu
  • Select Senders
  • View senders by number of threads
  • Search for a specific address
  • Sort the list alphabetically by address name
  • View senders by the size of their emails
  • Search for a specific address
  • Filter out only senders with a specific size of emails
  • Sort senders alphabetically by address
  • Press the trash icon to move a sender to the trash folder
  • Press the trash button in the popup to move all emails from this sender into your Gmail trash folder
  • Notice: emails are not deleted until you manually empty your trash folder, or they have been in the trash folder for more then 30 days.
  • Press the label icon
  • Enter a valid name for the label (or keep the one automatically chosen)
  • Select a category, or keep "No Category".
  • Press the label button to have all emails from the sender moved out of the inbox and into the label category/name provided.
